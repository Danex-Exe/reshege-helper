## ReshEge Helper

ReshEge Helper is a userscript that improves the ReshEge game interface by adding a modern menu, match history, leaderboard view, and customization options.

## ✨ Features

* Interactive and easy-to-use side menu
* Built-in leaderboard viewer
* Detailed match history with error insights
* Light and dark theme support
* Smooth and responsive interface
* Connection status indicator
* Extensible system for adding new features

## 📦 Installation

To use the script, install any userscript manager such as Tampermonkey or Violentmonkey. Then create a new script, paste the contents of the project file, and save it. After that, open ReshEge in your browser.

## 🚀 Usage

Once installed, a “Menu” button will appear in the interface. Through it, you can access leaderboard data, view your match history, and switch between themes.

## 📊 Match History

The match history shows key information about previous games, including opponents, score changes, and timestamps. It also provides a breakdown of mistakes with detailed context for each problem.

## 🎨 Themes

The script supports both light and dark themes. Your selected theme is saved automatically and applied on the next visit.

## ⚙️ Architecture

The project is structured around a modular menu controller responsible for rendering UI components and handling navigation. Data such as leaderboard and match history is received dynamically and displayed in the interface.

## 🔧 Customization

The script is designed to be extendable, allowing additional features or menu entries to be integrated without modifying the core logic.

## 🐛 Debugging

Basic initialization and status messages are available in the browser console to help diagnose issues.
